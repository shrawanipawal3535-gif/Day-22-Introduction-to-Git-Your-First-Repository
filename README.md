# Day-22-Introduction-to-Git-Your-First-Repository
Git is used to track and manage changes in your project.
First, configure your identity using git config --global user.name and git config --global user.email.
Create a folder, run git init to start a repository.
Use git add and git commit -m "message" to save your changes.
